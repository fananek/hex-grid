Structure Axial​Coordinates 

public struct AxialCoordinates: Codable

Represents coordinates in Axial format using q, r axis. q - you can imagine as a column r - you can imagine as a row

Axial coordinates are used mainly for storage purposes. Use it carefully because it can't be validated like Cube coordinates.
